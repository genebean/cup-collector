Updating the App

Because Cup Collector is deployed as a Nix flake, updating means bumping the flake input and rebuilding. There is no manual download or installation step.

On this page
  1. The normal update flow
  2. Checking what changed
  3. Rolling back
  4. When to update

The normal update flow

# In your NixOS config repository:

# 1. Bump the cup-collector flake input to latest
nix flake update cup-collector

# 2. Check what changed (optional but recommended)
git diff flake.lock

# 3. Rebuild and switch
sudo nixos-rebuild switch --flake .#your-hostname

Checking what changed

Before rebuilding, you can review the commits included in the update:

# Update the lock file and commit it in one step
nix flake update cup-collector --commit-lock-file

Or browse the commit history directly at github.com/genebean/cup-collector/commits/main.

Rolling back

If something breaks after an update, NixOS keeps the previous generation — you can roll back instantly:

sudo nixos-rebuild --rollback switch

Or switch to a specific earlier generation:

# List available generations
nix-env --list-generations --profile /nix/var/nix/profiles/system

# Switch to a specific generation number
sudo nix-env --switch-generation 42 --profile /nix/var/nix/profiles/system
sudo systemctl daemon-reload
sudo systemctl restart cup-collector

Rolling back restores the previous version of the app, its config, and all NixOS services. Your PocketBase data is not affected — only the running software changes.

When to update

  • Update promptly when release notes mention a security fix or a bug that affects you.
  • Update at leisure for new features — the app is designed for long-term stability.
  • Cup Collector has no external runtime dependencies that go stale (other than the Google Places API, which is versioned by Google independently of app releases).
Your collection data is not affected by app updates The PocketBase data lives at /var/lib/cup-collector/pb_data/. An app update only changes the Next.js server binary and static assets — your cups, photos, and household record are untouched.
