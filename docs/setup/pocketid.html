PocketID Setup

Register Cup Collector in PocketID and add each household member. This must be done before the app will work — authentication depends on it.

Step 1 — Register the application

  1. Log in to your PocketID admin panel (e.g. https://id.yourdomain.com → Admin).
  2. Navigate to Applications → Create Application.
  3. Fill in the fields as follows:

Logo / IconUpload public/icons/icon-192.png (192×192 PNG)
Post-logout redirecthttps://cups.yourdomain.com
  1. Save the application. PocketID will generate a Client ID and Client Secret.
  2. Copy both values — you'll need them for the .env secrets file.

Step 2 — Add users to PocketID

Each person who will use Cup Collector needs a PocketID account. For each person:

  1. In the PocketID admin panel, go to Users → Create User.
  2. Enter their name, email, and set a temporary password (they can change it on first login).
  3. Save. The user can now log in to PocketID.

Step 3 — Find each user's sub value

The household record in PocketBase uses each user's PocketID sub claim to determine their role. To find a user's sub:

  1. In the PocketID admin panel, go to Users.
  2. Click on the user. Their User ID field is their sub. Copy it.
  3. Keep a note of each person's sub — you'll use them when creating the household record in PocketBase.
Alternative: read it from the session
After deploying the app, a user can log in and check their sub via Settings → the app shows their role. If you need the raw sub value, it also appears in the PocketBase admin UI once the user has logged in and a household has been created.

Step 4 — Verify the "My Apps" launch icon

PocketID shows registered applications in each user's "My Apps" panel with a clickable icon. To verify this is working:

  1. Have a user log in to PocketID.
  2. They should see a "Cup Collector" tile in their apps panel with the icon you uploaded.
  3. Clicking the tile should open https://cups.yourdomain.com.

If the icon doesn't appear, check that the Launch URL is saved correctly and that the icon image uploaded without errors.
